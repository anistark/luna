# Luna ☾

![luna](https://github.com/user-attachments/assets/8e8a669c-3e04-4d76-8cb8-2f2308b0c2f0)

A TypeScript SDK for uploading files & directories to IPFS using Helia.

## Dev

```sh
npm i
```

### Build

```sh
npm run build
```

### All Test

```sh
npm run test
```

### Test Dev mode

```sh
npm run test:dev
```
